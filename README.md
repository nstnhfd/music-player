# Requirements
1. pyqt5
2. pygame
3. mutagen
4. random
5. time
6. sys
# Features
* Add musics to list
* play music one by one
* mute music
* unmute music
* volume up and down
* stop music 
* next music of list
* previouse music of list
* show time of played music by progressbar
